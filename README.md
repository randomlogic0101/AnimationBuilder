# AnimationBuilder
This application takes files from inkscape (with added tags in the xml editor) and creates an MP4 based on the animation descriptions

## Requirements

Currently you must have the following packages installed on your system:

- ffmpeg
- resvg

## Future

The plan is to move ffmpeg, and resvg to python libraries so there will be no reliance on system packages.
